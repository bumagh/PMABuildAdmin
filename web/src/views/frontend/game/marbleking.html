订阅常见问题

订阅后如何获取内容？

支付成功后，系统将引导您加入对应的专属微信社群。所有更新、资料链接、Demo下载码均在群内以公告形式发布。同时会绑定一个加密云盘，用于存放大型文件。

Demo可以一直玩吗？

Demo为阶段性测试版本，有固定的测试开放期（通常为2周）。测试结束后服务器关闭，但安装包您可以保留。

取消订阅后，已获得的内容会消失吗？

数字文件（原画、音乐、设定集）一旦发送，永久属于您。游戏内的虚拟物品（皮肤、称号）也会永久保留。但取消后将无法接收后续更新，并失去社群身份和访问权限。

游戏上线后，这个订阅会怎样？

游戏正式上线后，「创世议会」订阅计划将停止新用户加入。但对老订阅者，我们将转型为一个持续的 "幕后内容更新"频道，继续提供开发幕后、扩展包前瞻等内容，订阅费可能会调整，但老用户享原价续订权。

为什么值得订阅？

您不是在购买一个商品，而是在资助并参与一场创造。

• 见证生长：从线稿到动画，从旋律到史诗，您将看到一切如何从无到有。
• 塑造历史：您的反馈可能直接改变某个弹珠的轨迹，或是一个战场的样貌。
